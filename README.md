## 它能帮你做什么

Right Click Enhancer 是 RBSoft 出品的一款 Windows 右键菜单管理工具，核心能力就两件事：**往右键菜单里加东西，以及把不要的东西踢出去**。

听起来简单，但实际用起来你会发现这两个动作能解决很多日常麻烦。

**清理冗余菜单项**是它最立竿见影的功能。那些被各种软件强行塞进右键的"用 XX 打开""上传到 XX 云盘"之类的选项，你可以直接禁用或删除，菜单瞬间清爽。我清理完之后，右键菜单从一长串缩回到了四五项，看着就舒服。

**添加自定义快捷方式**则是另一面的玩法。你可以把常用文件夹、程序甚至脚本挂到右键菜单里。比如我做开发，把 hosts 文件、常用的配置脚本直接挂到右键，点一下就能用记事本打开，再也不用一层层翻目录找路径。

除了这两项核心功能，它还内置了一些实用的小工具：

- **批量重命名**：支持动态变量、搜索替换，还能提取图片的 Exif 信息或 MP3 标签来命名文件，处理一堆素材的时候特别省事
- **计算文件 Hash 值**：右键直接算 MD5、SHA，下载完大文件校验完整性不用再开额外工具
- **复制文件内容**：右键直接把文件内容复制到剪贴板
- **新建文件模板自定义**：不用只拘泥于系统自带的那几个"新建"选项
- **IE 右键菜单管理**：连浏览器里的右键都能一起收拾

对高级用户来说，它还支持给挂载的程序配置运行参数和变量，能玩出一些自动化调用的花样。
